将 HTML 报告、仪表盘和交互式决策表单发布到公网链接——完整覆盖安全校验、主题一致性和一键分享。

## What It Does

- **正确的文件路径** — 写到 nginx 服务的正确目录，避免 404
- **主题一致性** — 统一应用亮色主题 CSS，而非临时暗色样式
- **安全校验** — 公网可见前检查页面是否安全
- **自动发布** — 页面生成后立即可通过公网 URL 访问

## How to Use

直接让 agent 创建报告或 HTML 页面，它会自动处理发布流程。把生成的链接分享到任意平台即可。

## When Not to Use

如果是主站公开 SEO/博客页面，使用专门的 SSG 工作流——这个 skill 是为私有报告页面设计的。

## Why

没有这个 skill，agent 会把 HTML 写到随机目录、搞乱主题一致性、或者把带内部数据的页面直接暴露。这套流程让发布变得可靠且安全。
