# Web Research

## When to Use

Use this skill when the user asks for research, competitor analysis, technical investigation, or a source-backed summary.

## Workflow

1. Clarify the research question and desired output.
2. Search from at least three angles: official sources, community examples, and recent discussions.
3. Prefer primary sources over summaries.
4. Cross-check important claims across multiple sources.
5. Separate facts, interpretation, and recommendations.

## Quality Bar

- Cite source URLs for factual claims.
- Mark uncertain claims as unverified.
- Prefer recent sources when researching fast-moving topics.
- Keep the final answer structured and decision-oriented.

## Output Format

```markdown
## Key Findings
- ...

## Sources
- ...

## Recommendation
- ...
```
