# Daily Standup

Turn messy notes, commits, and chat context into a clean three-part standup update in seconds.

## What It Does

Takes whatever context you have — recent commits, task notes, Slack/Telegram messages, pasted bullets — and distills it into a concise standup covering:

1. **Yesterday** — what got done
2. **Today** — what's planned
3. **Blockers** — what's in the way

It strips implementation noise, keeps each section short, and never invents details. If context is missing, it calls out assumptions rather than fabricating progress.

## How to Use

Paste your notes, link to a task board, or just describe what you worked on. It will organize everything into the standup format.

## Output Example

```
Yesterday
- Merged PR #142: auth token refresh flow
- Reviewed #138 and #140
- Fixed notification dedup bug in production

Today
- Ship auth refresh to staging
- Start on billing dashboard v2 wireframes
- Pair with Derek on container health check

Blockers
- Waiting on API team to expose usage endpoint (ETA Thursday)
```

## Tips

- Works best with bullet-style input — paste your day's scratch notes directly
- If you're in a hurry, just list 3-5 bullets and it handles the structure
- Mentions teammates by name so your team knows who to follow up with
## Why Use This Skill

Writing standup updates from scratch every morning is a waste of brain cycles. This skill digests your commits, notes, and chat context into a clean three-part update ready to paste into Slack. It strips noise and never invents progress.
